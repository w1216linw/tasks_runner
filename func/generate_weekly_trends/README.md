# Monday Order Trend - 每周一订单趋势分析

## 项目简介

本项目用于分析和可视化每日订单数据，重点追踪每周一的订单量趋势。通过读取多个CSV订单文件，生成包含所有每日订单量的趋势图，并特别标注每个周一的数据点。

## 项目结构

```
monday_order_trend/
├── orders/                          # 订单数据目录
│   ├── orders_250915-250921.csv    # 周订单数据（9月15-21日）
│   ├── orders_250922-250928.csv    # 周订单数据（9月22-28日）
│   └── ...                          # 更多CSV文件
├── plot_monday_trend.py             # 主分析脚本
├── monday_order_trend.png           # 生成的趋势图（输出文件）
├── venv/                            # Python虚拟环境
└── README.md                        # 项目文档（本文件）
```

## 数据格式

### CSV文件命名规则
- **周数据**：`orders_YYMMDD-YYMMDD.csv`（例如：`orders_250915-250921.csv`）
- **单日数据**：`orders_YYMMDD.csv`（例如：`orders_251117.csv`）

### CSV文件内容格式
```csv
State_Code,State_Name,2025-09-15,2025-09-16,2025-09-17,...,Total
FL,Florida,6715,3308,2817,3099,...,23674
CA,California,4178,2330,2175,2067,...,15695
TX,Texas,3540,1945,1782,1903,...,14076
...
```

**列说明：**
- `State_Code`：州代码（如 FL, CA, TX）
- `State_Name`：州名称（如 Florida, California, Texas）
- `2025-MM-DD`：每一天的订单量列（日期格式：YYYY-MM-DD）
- `Total`：该州当周/日总订单量

## 使用方法

### 1. 添加新的订单数据

将新的订单CSV文件放入 `orders/` 目录中。文件需符合上述数据格式要求。

### 2. 运行分析脚本

```bash
python plot_monday_trend.py
```

### 3. 查看输出

脚本运行后会：
1. **控制台输出**：显示读取的数据统计和每个周一的订单量
   ```
   正在读取订单数据...
   ✓ 已读取 64 天的数据（10 个周一）

   每周一订单量：
     2025-09-15 (周一): 36,837 单
     2025-09-22 (周一): 41,637 单
     ...
   ```

2. **生成图表**：在项目根目录生成 `monday_order_trend.png`
   - 图表宽度：20英寸，高度：8英寸
   - 分辨率：300 DPI
   - 格式：PNG

## 图表说明

### 视觉元素

1. **数据点和折线**：
   - 蓝色折线连接所有每日订单量
   - 每个数据点显示当天订单数值

2. **周一标注**（蓝色框）：
   - 除最新周一外的所有周一用蓝色边框标注
   - 显示日期（月/日）和订单量

3. **最新周一标注**（红色框）：
   - 最新的周一数据用红色特别标注
   - 更大的标记点和加粗文字
   - 突出显示最新趋势

4. **X轴**：
   - 只显示每个周一的日期（月/日格式）
   - 非周一日期不显示，避免拥挤

5. **Y轴**：
   - 订单量，使用千位分隔符格式化

## 脚本功能详解

### 主要函数

#### `extract_all_daily_orders()`
- **功能**：从 `orders/` 目录中读取所有CSV文件
- **处理**：提取每一天的订单总量（所有州求和）
- **返回**：包含日期、订单量、是否为周一的DataFrame

#### `plot_trend(df)`
- **功能**：生成每日订单趋势图
- **特点**：
  - 自动识别周一并特别标注
  - 最新周一用红色突出显示
  - 非周一数据点显示数值但不加框
  - X轴只显示周一日期

#### `main()`
- **功能**：主控制函数
- **流程**：
  1. 读取并处理所有订单数据
  2. 输出统计信息
  3. 生成趋势图

## 常见问题

### Q1: 中文显示乱码怎么办？
**A**: 脚本已配置中文字体支持，优先使用：
- SimHei（黑体）
- Microsoft YaHei（微软雅黑）
- Arial Unicode MS

如果仍有问题，请确保系统已安装这些字体。

### Q2: 如何修改图表样式？
**A**: 在 `plot_monday_trend.py` 中可以修改：
- 第52行：`figsize=(20, 8)` - 调整图表尺寸
- 第63行：`color='#2E86AB'` - 修改折线颜色
- 第140行：`dpi=300` - 调整图表分辨率

### Q3: 数据文件格式错误怎么办？
**A**: 确保CSV文件：
- 使用UTF-8编码（支持UTF-8-BOM）
- 日期列格式为 `YYYY-MM-DD`
- 包含 `State_Code` 和 `State_Name` 列
- 每行数据完整无缺失

### Q4: 如何只分析特定日期范围？
**A**: 可以修改 `extract_all_daily_orders()` 函数，在第20行后添加文件过滤逻辑。
